# KULemma

A Bangla lemmatizer developed by DGTED Lab, Khulna University.

*Accuracy*: 96.90.

### Development Steps:
- Compiling lingual resources.
- Designing algorithms.
- Assembling stemming rules.
- Implementing lemmatizer.
